## Manual

If you wish to install `gluestack-ui` with nativewind into your existing project, you can proceed with the following steps:

### Step 1: Setup NativeWind

To setup Expo project using `NativeWind` follow [NativeWind documentation](https://www.nativewind.dev/v4/getting-started/react-native).

### Step 2: Install Dependencies

Install the dependencies of gluestack-ui in your project. This can be done using the following command:

```bash
yarn add @gluestack-ui/nativewind-utils

```

OR

```bash
npm i @gluestack-ui/nativewind-utils

```

### Step 4: Configure components path

Create a `components/ui` folder inside `src` folder and add path in `tsconfig.json`

```js

  "paths": {
    "@/*": ["./src/*"] // Add relative path
  },

```

### Step 5: Configure GluestackUIProvider

To add config, create a `GluestackUIProvider/config.ts` file in your `components/` folder and paste the following code.

<CollapsibleCode>

```jsx
%%-- File: core-components/nativewind/gluestack-ui-provider/config.ts --%%
```

</CollapsibleCode>

To add `GluestackUIProvider`, create a `GluestackUIProvider/index.tsx` file inside `components/` folder and paste the following code.

<CollapsibleCode>

```jsx
%%-- File: core-components/nativewind/gluestack-ui-provider/index.tsx --%%
```

</CollapsibleCode>

### Step 6: Usage

Wrap your app with `GluestackUIProvider` in `App.tsx`.

```jsx
// App.tsx
export default function App() {
  return (
    <GluestackUIProvider>{/* Add your app code here */}</GluestackUIProvider>
  );
}
```
