# Getting Started

## Automatic (nativewind)

### Using Template

you can use [gluestack-expo-nativewind-v4-template](https://github.com/gluestack/gluestack-expo-nativewind-v4-template).

## Manual (nativewind)

To setup expo project using nativewind follow nativewind [docs](https://www.nativewind.dev/v4/getting-started/react-native).

### Step4

Adding Theme Provider

Create a `ThemeProvider.tsx` file inside `components/core` folder

```jsx
// ThemeProvider.tsx

import { themes } from '@/components/core/theme';
import React from 'react';
import { View } from 'react-native';

export function ThemeProvider({
  mode,
  ...props
}: {
  mode?: 'light' | 'dark',
  children: any,
}) {
  return (
    <View
        style={[
          mode ? config[mode] : config['light'],
          providerStyle,
          props.style,
        ]}
      >
       {props.children}
    </View>
  )
}
```

### Step5

Adding Theme Provider to App.tsx

```jsx
// App.tsx

export default function App() {
  return (
    <ThemeProvider mode="light">{/* Add your app code here */}</ThemeProvider>
  );
}
```
