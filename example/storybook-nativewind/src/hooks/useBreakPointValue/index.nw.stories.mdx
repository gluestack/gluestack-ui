## API Reference

To use this component in your project, include the following import statement in your file.

```jsx
import { useBreakPointValue } from '@/components/hooks/use-break-point-value';
```

```jsx
const flexDir = useBreakpointValue({
                default: "column",
                sm: "row",
              });
```
