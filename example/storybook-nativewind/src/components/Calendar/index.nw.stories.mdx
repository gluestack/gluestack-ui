## API Reference

To use this component in your project, include the following import statement in your file.

```jsx

import {
  Calendar,
  CalendarHeader,
  CalendarHeaderNext,
  CalendarHeaderPrev,
  CalendarHeaderTitle,
  CalendarContent,
  CalendarWeek,
  CalendarDays,
} from '@/components/ui/calendar';

```
```jsx
export default () => (
  <Calendar
    value={new Date()}
    onChange={(date) => console.log(date)}
  >
    <CalendarHeader>
      <CalendarHeaderPrev />
      <CalendarHeaderTitle />
      <CalendarHeaderNext />
    </CalendarHeader>
    <CalendarContent>
      <CalendarWeek />
      <CalendarDays />
    </CalendarContent>
  </Calendar>
);
```

#### Calendar with Custom Renderer
Here is a complete example of how to use the `Calendar` component with its `render` prop to customize the appearance of the calendar.

<Wrapper>
  <CodePreview
    showComponentRenderer={true}
    showArgsController={false}
    metaData={{
      code: `
        <Calendar
          value={new Date()}
          className="w-72"
          minDate={new Date(2024, 5, 1)}
          maxDate={new Date(2025, 1, 15)}
          onChange={(day) => console.log(day)}
        >
          <CalendarHeader>
            <CalendarHeaderPrev />
            <CalendarHeaderTitle />
            <CalendarHeaderNext />
          </CalendarHeader>
          <CalendarContent>
            <CalendarWeek />
            <CalendarDays
            render={({ day, ...dayProps }) => (
              <Pressable
                key={day?.getDate()}
                {...dayProps}
                className="w-[14.28%] p-2 flex items-center justify-center rounded-lg"
              >
                <Text>{day?.getDate()}</Text>
              </Pressable>
            )}
            />
          </CalendarContent>
        </Calendar>
      `,
      transformCode: (code) => {
        return transformedCode(code);
      },
      scope: { Calendar, CalendarHeader, CalendarHeaderPrev, CalendarHeaderNext, CalendarHeaderTitle, CalendarContent, CalendarWeek, CalendarDays, Wrapper, Pressable, Text },
      argsType: {},
    }}
  />
</Wrapper>

#### Default Usage with CalendarDate
The default implementation uses `CalendarDate` internally:

```jsx
<Calendar>
  <CalendarContent>
    <CalendarWeek />
    <CalendarDays /> 
  </CalendarContent>
</Calendar>
```

#### Custom Implementation with CalendarDate
You can provide your own implementation while still using `CalendarDate`.

```jsx
<CalendarDays
  render={({ day, ...dayProps }) => (
    <CalendarDate date={day} {...dayProps}>
      {/* Custom content inside CalendarDate */}
      <Text>{day?.getDate()}</Text>
    </CalendarDate>
  )}
/>
```

#### Custom Implementation
Or create a completely custom date cell:

```jsx
<CalendarDays
  render={({ day, ...dayProps }) => (
    <Box
      {...dayProps}
      className="w-[14.28%] p-2 flex items-center justify-center"
    >
      <Text>{day?.getDate()}</Text>
    </Box>
  )}
/>
```
