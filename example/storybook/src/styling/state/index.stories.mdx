## States

Styling of gluestack-ui components is done with help of [gluestack-style](https://style.gluestack.io/).
gluestack-style provides different features that help us to manage state, color mode, responsiveness and much more. Right now we will discuss how does gluestack style helps us to manage states like hover, active, focused etc.

Easiest way to change styling of states would be using sx prop which enables state management.

<AppProvider>
  <CodePreview
    showComponentRenderer={true}
    showArgsController={false}
    metaData={{
      code: `
      <Button sx = {{ ":hover": {
        backgroundColor:"$red500"
        }
      }}>
        <ButtonText>Hello world</ButtonText>
      </Button>
      `,
      transformCode: (code) => {
        return transformedCode(code);
      },
      scope: {
        Wrapper,
        Text,
        Button,
        ButtonText,
      },
      argsType: {},
    }}
  />
</AppProvider>

<br />

We can define styles that can be applied on different states and pass them as an object in sx prop.

### Supported States are

- indeterminate
- checked
- readOnly
- required
- invalid
- focus
- focusVisible
- hover
- pressed
- active
- loading
- disabled
