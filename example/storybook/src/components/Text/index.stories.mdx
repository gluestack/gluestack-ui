# Installation

This command copies the `Text` component to your project.

```jsx
npx install @gluestack/ui add Text
```

# Advanced

## Customizing the Text

For customizing the Text, you have to map styled view to Text. You can refer [dank.style](https://dank.style/) for more information on how to use styled components.

### Usage

```jsx
// import the styles
import { Root } from './components/Text';

// mapping the Text to root component
const Text = Root;

// Using the text component
export default () => <Text />;
```

Default styling of the component can be found in the `components/Text` file. For reference, you can view the [source code](https://github.com/gluestack/universa11y/blob/development/example/storybook/src/components/styled-components/text/index.tsx) of the styled `Text` component.
