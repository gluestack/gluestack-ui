## API Reference

### Import

To use this component in your project, include the following import statement in your file.. This will import the component from your components folder.

```jsx
import {
  AlertDialog,
  AlertDialogBackdrop,
  AlertDialogContent,
  AlertDialogHeader,
  AlertDialogCloseButton,
  AlertDialogFooter,
  AlertDialogBody,
} from '@gluestack-ui/themed';
```

### Anatomy

The structure provided below can help you identify and understand a AlertDialog component's various parts.

```jsx
export default () => (
  <AlertDialog>
    <AlertDialogBackdrop />
    <AlertDialogContent>
      <AlertDialogHeader>
        <AlertDialogCloseButton />
      </AlertDialogHeader>
      <AlertDialogBody />
      <AlertDialogFooter />
    </AlertDialogContent>
  </AlertDialog>
);
```
