## API Reference

### Import

To use the component in your project, run the command below into your import statement. This will import the component from your components folder.

```bash
import { HStack } from '@gluestack-ui/themed';
```

### Anatomy

The structure provided below can help you identify and understand a HStack component's various parts.

```jsx
export default () => <HStack />;
```

### Component Props

This section provides a comprehensive reference list for the component props, detailing descriptions, properties, types, and default behavior for easy project integration.

#### HStack

It inherits all the properties of React Native's [View](https://reactnative.dev/docs/view) component.

### Accessibility

The accessibility of a HStack is primarily determined by the accessibility information of the components it contains. When you pass an accessible component inside a HStack, its accessibility attributes, such as labels and hints, will be utilized by assistive technologies like screen readers.

## Themed

The themed version of the component is a pre-styled version of the component, which allows you to quickly integrate the component into your project. The component's design and functionality are fully defined, allowing you to focus on the more important aspects of your project.
