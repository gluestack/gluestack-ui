# Installation

This command copies the `IconButton` component to your project.

```jsx
npx install @gluestack/ui add IconButton
```

## Usage

```jsx
// import the component from your component folder
import { IconButton } from 'components';

// Using the icon-button component
export default () => (
  <IconButton>
    <IconButton.Text>IconButton</IconButton.Text>
  </IconButton>
);
```

## Anatomy

The IconButton component consists of the following parts:

```jsx
export default () => (
  <IconButton>
    <IconButton.Text>IconButton</IconButton.Text>
    <IconButton.Spinner />
  </IconButton>
);
```

## API Reference

Contains all icon-button related layout style props and actions.
It inherits all the properties of React Native's [Pressable](https://reactnative.dev/docs/pressable#props) component.

### IconButton.Text

Contains all text related layout style props and actions.
It inherits all the properties of React Native's [Text](https://reactnative.dev/docs/text#props) component.

### IconButton.Spinner

Contains all spinner related layout style props and actions.
It inherits all the properties of React Native's [ActivityIndicator](https://reactnative.dev/docs/activityindicator#props) component.

## Features

- Keyboard support for actions.
- Support for hover, focus and active states.
- Option to add your styles or use the default styles.

## Accessibility

### Keyboard

- Tab: Moves focus to the next focusable element.
- `Shift + Tab`: Moves focus to the previous focusable element.
- Enter: Triggers the icon-button's action.

### Screen Reader

- VoiceOver: When the icon-button is focused, the screen reader will announce the icon-button's label and its current state.

## Dependencies

- `@gluestack-ui/icon-button`
- `@gluestack-ui/utils`
- `@react-native-aria/focus`
- `react-native-svg`

# Advanced

## Customizing the IconButton

We have a function called `createIconButton` which can be used to create a custom icon-button component. This function takes in a configuration object which contains the styled components that you want to use for the icon-button. You can refer [dank.style](https://dank.style/) for more information on how to use styled components.

### Usage

```jsx
// import the styles
import {
  Root,
  Text,
  Group,
  GroupHSpacer,
  GroupVSpacer,
  Spinner,
} from './components/IconButton';

// import the createIconButton function
import { createIconButton } from '@gluestack-ui/icon-button';

// Understanding the API
const IconButton = createIconButton({
  Root,
  Text,
  Group,
  GroupHSpacer,
  GroupVSpacer,
  Spinner,
});

// Using the icon-button component
export default () => (
  <IconButton.Group>
    <IconButton>
      <IconButton.Text />
      <IconButton.Spinner />
    </IconButton>
  </IconButton.Group>
);
```

Default styling of all these components can be found in the `components/IconButton` file. For reference, you can view the [source code](https://github.com/gluestack/universa11y/blob/development/example/storybook/src/components/styled-components/icon-button/index.tsx) of the styled `IconButton` components.
